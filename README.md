# dotfiles (for debian/ubuntu)

## Setup (for GitHub Codespaces Default Linux Universal base image)

```
git clone https://github.com/ToruMakabe/dotfiles.git ~/dotfiles
cd ~/dotfiles
./bootstrap.sh
```

## Setup (Link only)

```
git clone https://github.com/ToruMakabe/dotfiles.git ~/dotfiles
cd ~/dotfiles
./bootstrap.sh link-only
```
